Присваивание

Оператор присваивания =

Оператор = записывает значение из правой части в переменную слева. Это самая базовая операция в C++. Также существуют составные варианты: +=, -=, *=, /= — они сокращают запись при обновлении значений.


Присваивание возвращает значение, поэтому возможны цепочки: a = b = c = 0;

int a = 5;       // базовое
int b = 3;

a += b;          // a = a + b → 8
a -= 2;          // a = a - 2 → 6
a *= 2;          // a = a * 2 → 12
a /= 4;          // a = a / 4 → 3

// цепочка
int x, y, z;
x = y = z = 0;

Тернарный

Тернарный оператор ? :

Краткая форма if/else: вычисляет условие, и если оно истинно — возвращает первое значение, иначе второе. Удобен для коротких условных присваиваний.


Синтаксис: условие ? если_да : если_нет


⚠️ Не злоупотребляйте вложенными тернарными — код теряет читаемость.

int a = 10, b = 20;

// классический if/else
int max;
if (a > b) max = a;
else       max = b;

// тернарный — то же самое
int max2 = (a > b) ? a : b; // 20

// в выводе
std::cout << (a > 0 ? "pos" : "neg");
⚙️
Составное присваивание

Сокращённая запись

Операторы +=, -=, *= вычисляют и присваивают в одну строку — читаемее и быстрее для компилятора.

int score = 100; score -= 10; // штраф score += 50; // бонус // score = 140
🔁
Вложение

Тернарный в условиях

Тернарный оператор возвращает значение, поэтому его можно вставлять прямо в выражения и функции.

int x = -5; int abs_x = (x < 0) ? -x : x; std::cout << "Модуль: " << abs_x; // 5
💡
Совет

Когда что использовать?

Используй тернарный для простых условных значений. Если логика сложнее двух веток — лучше if/else.

// ✅ Хорошо — просто string s = n > 0 ? "+" : "-"; // ❌ Плохо — нечитаемо int r = a ? b ? c : d : e;
